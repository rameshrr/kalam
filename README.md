
# utils
Utility library for node.js

# Installation
install via [NPM](https://www.npmjs.com/):
> npm install utils

# Usage
### Initializing:
```javascript

var utils = require('utils');

```

### getUid([options]);
```javascript

var uid = utils.getUid();
utils.log(uid);

```
